Quote for Ahau for Te manaaki i te ahurea tuakiri

     

2020-08-18

       

Our estimate for delivering this project plan is $266,800 (NZD, ex gst).

      
       

As we’ve discussed, this is an estimate, because we follow an agile methodology. This means that the size and nature of each part of your plan can be adapted to as new constraints emerge. It also means that is a particular piece turns out to be simpler than expected, we can pass on those savings, or choose to focus those extra resources on additional features.

       

Please find attached an itemised breakdown of our estimates for each section of this plan. Project management and coordination are factored into each item. The “low estimate” numbers are the lowest we think the work could possibly cost to deliver. It’s usual for one or two features to be more straightforward (and therefore cheaper), but not all of them. The high estimate is a guarenteed maximum cost that we will commit to charging you for all of the features discussed in the project plan “Te manaaki i te ahurea tuakiri”.

       

If the project plan changes substantively (such as changes in behaviour / design / technical requirements, or the addition of new features), we will need to update our estimates and contract. Small are accounted for in the current estimate.

This quote is valid for 60 days.
